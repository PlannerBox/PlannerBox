## Description

API creating communication between front & database  
Using [Nest](https://github.com/nestjs/nest) framework with TypeORM

## Installation

```bash
$ npm i
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
